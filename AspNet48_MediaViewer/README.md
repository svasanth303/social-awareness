# ASP.NET 4.8 Media Viewer

This is a simple ASP.NET Web Forms project (.NET Framework 4.8) that shows:

- ✅ Online video playback
- ✅ Static photo gallery

## How to Run

1. Open in Visual Studio 2019 or 2022.
2. Build the project.
3. Add your media files to `/media/`.
4. Press F5 to launch the site.

## Pages

- `/Video.aspx` – Video Player
- `/Gallery.aspx` – Image Gallery
